Expenzo is a full-stack MERN (MongoDB, Express, React, Node.js) application designed to help users track, manage, and analyze their daily expenses.
It provides a clean UI, secure authentication, and a fully functional backend API for seamless expense management.

Features

Add, edit, and delete expenses

Categorize expenses (food, travel, utilities, etc.)

Monthly and overall spending summary

JWT-based authentication

MongoDB database for scalable, document-based storage

Responsive UI built with React + TypeScript

RESTful API using Node.js + Express

Deployed frontend and backend on Vercel / AWS (optional)
